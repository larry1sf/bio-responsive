Tener cuenta para seleccionar las actividades que ya se allan realizado en el dia.

Actividad Hora de registro Hora de finalizacion Checkeo

instrucciones

Selecciona la hora de inicio y finalizacion de registro para almacenarlo en la base de datos.

Debe marcar la casilla de "Realizado" para guardar la informacion y llevar una cuenta con el orden de los objetivos del dia.

Pulsar el boton "Añadir registro" para añadir una nueva fila a la tabla y poder rellenarla con la informacion deseada para su administracion.

El boton de "Enviar de datos" almacena los datos registrados en la tabla para poder observarlos en la pagina que se creara con la fecha estimada.
